Class Program

Creating a WebGL program for shaders is painful. This class simplifies the process.

class

Program

Object properties starting with $ are WebGL uniforms or attributes. Uniforms behave as expected: you can read/write a value. Attributes when read, return the location. And when written, enable/disabled this attribute. So you read integers and writte booleans.

param

WebGL context.

param

Object with two mandatory attributes: vert for vertex shader and frag for fragment shader.

param

(optional) If defined, the #include foo directives of shaders will be replaced by the content of includes.foo.
